Installation

Requirements

  • Python 3.11 or higher

  • uv package manager (recommended)

Install (recommended)

Use uv to create the project virtual environment and install dependencies for development:

uv sync --all-extras

From Source

Clone the repository and install in development mode:

git clone https://github.com/yourusername/secsuite.git
cd secsuite
uv sync --all-extras

This will install the package with all development dependencies.
