secsuite.example(items, *, output_path=None)[source]

Process a sequence of items and optionally write to a file.

Parameters:

  • items (Sequence[str]) – Sequence of string items to process.

  • output_path (Path | None) – Optional path to write results to.

Return type:

list[str]

Returns:

List of processed items in uppercase.

Raises:

  • ValueError – If items is empty.

  • OSError – If writing to output_path fails.

Examples

>>> example(["hello", "world"])
['HELLO', 'WORLD']
